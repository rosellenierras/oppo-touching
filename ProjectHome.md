Open source project to deliver JavaScript library SDK for UI development in touch devices.

**V2.0**
  * Fix some bugs;
  * Add sample code;
  * Support Android 3.1 or high version, see [wiki](http://code.google.com/p/oppo-touching/wiki/AndroidSupport);

**V1.0**
  * Work with Sencha Touch, move ext js 4 charting component to touch SDK;
  * Support SVG currently, also available soon on Canvas, VML support;
  * Check out samples according to [wiki](http://code.google.com/p/oppo-touching/wiki/SetupChartingSample)
  * It is totally free.
